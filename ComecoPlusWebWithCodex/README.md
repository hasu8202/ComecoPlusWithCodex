# コメ粉プラス 新Webサイト案

PLUS株式会社の既存サイトをもとに制作した、提案用の静的プロトタイプです。

## 見方

`index.html` をブラウザで開くと表示できます。インターネット接続中は、既存サイトの商品画像とGoogle Fontsが読み込まれます。

## ファイル

- `index.html`：ページ本文
- `styles.css`：デザインとレスポンシブ表示（生成した田園コンセプト画像を内包）
- `script.js`：スマートフォンメニューと表示アニメーション
- `提案・料金プラン.md`：お父さまへの提案方法、価格、保守範囲

## 現在の位置づけ

これは公開前の提案用初稿です。商品購入・お問い合わせボタンは既存サイトにつながります。本番公開前に、最新情報・アレルギー表記・認証・商品情報をPLUS株式会社へ確認し、実際の写真へ差し替えます。
